LAB 15 — Static NAT + Dynamic NAT (Enterprise-Style)
Objective
Configure Static NAT with 1:1 mapping for LAN1 PCs
Configure Dynamic NAT with a pool for LAN2 PCs
Use point-to-point router links for transit
Access server LAN behind R2
Static routing between all routers

Static NAT (R1)
Public Mappings
Private	Public
192.168.1.10	200.200.200.10
192.168.1.20	200.200.200.20

interface g0/0
 ip nat inside

interface g0/1
 ip nat outside

ip nat inside source static 192.168.1.10 200.200.200.10

ip nat inside source static 192.168.1.20 200.200.200.20

Dynamic NAT (R2)

Pool:

100.100.100.1
to
100.100.100.10
interface g0/0
 ip nat inside

interface g0/1
 ip nat outside

access-list 1 permit 192.168.2.0 0.0.0.255

ip nat pool DYN_POOL 100.100.100.1 100.100.100.10 netmask 255.255.255.0

ip nat inside source list 1 pool DYN_POOL

NO NAT.
