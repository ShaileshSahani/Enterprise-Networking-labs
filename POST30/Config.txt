🔧 LAB 30/50 — Enterprise WAN Through ISP Core
Objective

Build a simulated Internet/ISP cloud using OSPF.

HOME Router uses Static Routing
ISP EDGE redistributes static routes into OSPF
ISP Core runs OSPF Area 0
GOOGLE EDGE advertises Web Server network
End-to-End reachability from PC → Web Server

HOME Router
enable
conf t

hostname HOME

interface g0/0
 ip address 192.168.1.1 255.255.255.0
 no shutdown

interface s0/3/0
 ip address 49.1.1.1 255.255.255.252
 clock rate 64000
 no shutdown

ip route 0.0.0.0 0.0.0.0 49.1.1.2

end
wr
ISP EDGE
enable
conf t

hostname ISP-EDGE

interface s0/3/0
 ip address 49.1.1.2 255.255.255.252
 no shutdown

interface g0/0
 ip address 50.1.1.1 255.255.255.252
 no shutdown

ip route 192.168.1.0 255.255.255.0 49.1.1.1

router ospf 1
 router-id 10.10.10.10

 network 50.1.1.0 0.0.0.3 area 0

 redistribute static subnets

end
wr
