🔥 LAB 23 — OSPF Multi-Area Enterprise Expansion
🎯 Objective

Build a scalable enterprise OSPF design with:

Area 1 ---- Area 0 ---- Area 2

Verify:

Intra-area routing
Inter-area routing
ABR functionality
Route propagation through Area 0
📌 Topology
LAN A                         LAN B

192.168.1.0/24          192.168.4.0/24
    |                          |
    R1 ------- R2 ------- R3 ------- R4
         A1        A0       A2
IP Addressing
Device	Interface	IP
R1	G0/0	192.168.1.1/24
R1	G0/1	10.1.12.1/30
R2	G0/0	10.1.12.2/30
R2	G0/1	10.1.23.1/30
R3	G0/0	10.1.23.2/30
R3	G0/1	10.1.34.1/30
R4	G0/0	10.1.34.2/30
R4	G0/1	192.168.4.1/24
Area Design
R1-R2 = Area 1

R2-R3 = Area 0

R3-R4 = Area 2
Router Roles
R2 = ABR

Area 1
↕
Area 0

R3 = ABR

Area 0
↕
Area 2
🔷 R1 Configuration
enable
conf t

hostname R1

interface g0/0
 ip address 192.168.1.1 255.255.255.0
 no shut

interface g0/1
 ip address 10.1.12.1 255.255.255.252
 no shut

router ospf 1
 router-id 1.1.1.1

 network 192.168.1.0 0.0.0.255 area 1
 network 10.1.12.0 0.0.0.3 area 1

end
wr
